## ✨ Platform Features

- 🚀 **Extreme Performance**: Built on Astro 5's Islands Architecture for near-zero JS payloads.
- 🧠 **AI Content Engine**: Integrated workflows for AI-assisted technical writing and course generation.
- 🌍 **Triple-Locale Support**: Full native support for English, Traditional Chinese (繁體), and Simplified Chinese (简体).
- 🔓 **Secure Member Area**: Lightweight, modern authentication using **Better Auth** (Passkeys/2FA ready).
- 🌓 **Premium UI/UX**: Dark-mode first design with glassmorphism and motion-fluid interactions.

## 🏗️ Technical Stack (2026 Edition)

- **Foundation**: [Astro 5](https://astro.build/) (Static Site Generation + Server Islands)
- **Styling**: Vanilla CSS + Tailwind Core (Brand Design System V3)
- **Auth**: [Better Auth](https://better-auth.com/) (The 2026 standard for TS-native security)
- **Database**: Astro DB (libSQL/Turso) for lightweight, distributed state.
- **Content**: MDX & Content Collections with advanced Zod validation.

## 📦 Developer Setup

To run a local instance of the platform:

1. **Clone & Install**:
   ```bash
   git clone https://github.com/txs/andystv-web.git
   cd andystv-web
   npm install
   ```

2. **Launch Dev Server**:
   ```bash
   npm run dev
   ```
   *Access at `http://localhost:4321`*

3. **Build Version**:
   ```bash
   npm run build
   ```

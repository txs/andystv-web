# 9-3 補充 Alert, prompt, confirm

Original: https://course.andys.pro/Javascript-Intro/09-3-supplement-alert-prompt-confirm/

### 講義

#### 1. 警告視窗：`alert()`
這是最基本的對話框，用來顯示訊息給使用者，使用者只能點選「確定」關閉視窗。
*   範例：`alert("Hello World");`

#### 2. 確認視窗：`confirm()`
用來詢問使用者「是/否」或「確定/取消」。
*   **回傳值**：
    - 點擊「確定」：回傳 **`true`**。
    - 點擊「取消」：回傳 **`false`**。
*   **應用場景**：刪除資料前的再次確認、登出確認等。
*   範例：
    ```javascript
    var isSure = confirm("你確定要執行嗎？");
    if (isSure) {
        console.log("使用者按下確定");
    } else {
        console.log("使用者不想執行");
    }
    ```

#### 3. 輸入視窗：`prompt()`
提供一個文字方塊讓使用者輸入資訊。
*   **回傳值**：
    - 使用者輸入的內容（字串）。
    - 點擊「取消」或直接關閉：回傳 **`null`**。
*   範例：
    ```javascript
    var nickname = prompt("請輸入您的暱稱");
    if (nickname) {
        alert("嗨，" + nickname + "！");
    } else {
        alert("您沒有輸入暱稱。");
    }
    ```

#### 4. 小技巧：邏輯判斷
*   可以使用 **`!!` (雙驚嘆號)** 將 prompt 回傳的值快速轉為布林值來判斷是否有輸入。
*   結合 **三元運算子** 可以縮短程式碼長度：
    `nickname ? alert("嗨 " + nickname) : alert("沒輸入暱稱");`

# 9-3 如何使用 For/in 迴圈與 For/of 的差別

Original: https://course.andys.pro/Javascript-Intro/09-3-how-to-use-the-difference-between-forin-loop-and-forof/

### 講義

#### 1. For/in 迴圈：走訪「索引/鍵」
`for...in` 主要用來列舉物件的屬性名稱（Key）或是陣列的索引（Index）。
*   **用於陣列**：取得的是 `0, 1, 2...` 等索引。
    ```javascript
    var arr = [10, 20, 30];
    for (var i in arr) {
        console.log(i); // 印出 0, 1, 2
        console.log(arr[i]); // 印出 10, 20, 30
    }
    ```
*   **用於物件**：取得的是屬性名稱。
    ```javascript
    for (var key in person) {
        console.log(key + ": " + person[key]);
    }
    ```

#### 2. For/of 迴圈：走訪「值」
`for...of` 是 ES6 新增的語法，它會直接取出集合裡面的「內容物」。
*   **用於陣列**：
    ```javascript
    var arr = [10, 20, 30];
    for (var value of arr) {
        console.log(value); // 直接印出 10, 20, 30
    }
    ```
*   **限制**：`for...of` **不能**直接用於普通物件之上，因為物件不具備可疊代性（Iterable）。

#### 3. 如何對物件使用 For/of？
如果一定要用 `for...of` 處理物件，可以配合 `Object.keys()` 方法：
1. `Object.keys(person)`：會回傳一個包含所有鍵名的「陣列」。
2. 對該陣列使用 `for...of` 即可。
   ```javascript
   var keys = Object.keys(person); // ["name", "age", ...]
   for (var k of keys) {
       console.log(person[k]);
   }
   ```

#### 4. 總結
*   **想拿 Key** 或是要處理 **物件** ➝ 使用 `for...in`。
*   **想直接拿 Value** 或是處理 **陣列** ➝ 使用 `for...of`。

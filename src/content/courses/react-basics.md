# React 基礎入門課程

## 課程簡介

這是一門完全免費的 React 入門課程，適合：

- 熟悉 HTML/CSS/JavaScript 的初學者
- 想從 jQuery 轉換到現代框架的開發者
- 需要複習基礎的 React 開發者

## 📚 課程大綱

### 第一章：環境設定
- Node.js 安裝
- 建立第一個 React 專案
- VS Code 擴充套件推薦

### 第二章：JSX 語法
- JSX 基礎
- 條件渲染
- 列表渲染

### 第三章：元件開發
- 函數元件
- Props 傳遞
- 事件處理

### 第四章：State 管理
- useState Hook
- 表單處理
- 狀態提升

### 第五章：副作用
- useEffect Hook
- 資料獲取
- Cleanup 函數

### 第六章：實戰專案
- 打造 Todo App
- API 整合
- 部署上線

## 🎯 學習目標

完成本課程後，你將能夠：

✅ 理解 React 核心概念
✅ 獨立開發簡單的 React 應用
✅ 使用 Hooks 管理狀態
✅ 從 API 獲取資料並展示

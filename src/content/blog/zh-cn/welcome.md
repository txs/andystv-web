## 为什么选择 Astro？

这个网站使用 **Astro 4** 框架建立，优点包括：

1. **极致性能**：静态生成 + 部分 Hydration
2. **Markdown 原生**：AI 辅助写作超顺畅
3. **弹性架构**：公开内容静态，会员内容动态

```javascript
// 范例：Astro Content Collections
import { getCollection } from 'astro:content';

const posts = await getCollection('blog');
```
